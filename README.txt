SERA-Forensics as a solution to deploy digital forensic exercises built based on the information available of the students, the class and the whole context of the activities proposed during the lesson.
This solution can be integrated with SERA: https://www.nics.uma.es/pub/papers/1835.pdf
This research has been partially supported by the Innovative Education Projects PIE17-137 and PIE19-183 from the University of Malaga.

The main class is "seraforensics.py". You can try this code by using the main method. Some features depends on previous registration with some external platforms.
